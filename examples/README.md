# Примеры выполненных проектов команд студентов

Эта папка содержит образцы завершенных проектов, которые можно использовать в качестве эталона для оценивания и демонстрации.

## Структура примеров

### Python проекты
- `example-game-team/` - Завершенная игра "Угадай число"
- `example-calculator-team/` - Калькулятор с историей

### Web проекты
- `example-portfolio-team/` - Сайт-портфолио команды

## Использование

Преподаватели могут использовать эти примеры для:
- Демонстрации ожидаемого результата
- Оценивания качества студенческих проектов
- Показа лучших практик Git workflow

## Особенности примеров

Каждый пример содержит:
- ✅ Качественный README.md
- ✅ Историю коммитов с осмысленными сообщениями
- ✅ Правильную структуру проекта
- ✅ Рабочий код без ошибок
- ✅ Примеры Pull Request и код-ревью
